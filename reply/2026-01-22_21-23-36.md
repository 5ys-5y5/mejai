# 답변
토큰 없이 콘솔에서 호출이 불가능한 구조라서, **대시보드에서 버튼으로 시뮬레이션을 생성**하도록 연결했습니다. 이제 로그인 상태에서 버튼 클릭으로 바로 데이터가 생성됩니다. (Authorization 헤더는 내부적으로 처리됨)

변경 내용
- /app 대시보드에 “통화 시뮬레이션 생성” 버튼 추가
- 버튼 클릭 시 /api/simulate-call 호출 → 생성 후 자동 재조회
- 체크리스트 상세 항목 업데이트

테스트 방법
1) 서버 재시작: npm run dev
2) http://localhost:3000/app 접속
3) 우측 상단에 있는 “통화 시뮬레이션 생성” 버튼 클릭
4) 아래 항목 확인
   - 통화 수/이관률/가장 많이 호출된 에이전트 갱신
   - /app/calls에 신규 세션 추가
   - /app/calls/{id}에서 턴/ASR/요약/오디오 구간 표시
   - /app/review에 후속 지원 요청 생성(escalted:true)

변경 파일
- src/app/app/page.tsx
- docs/단계별_구현_체크리스트_및_검증.md

원하시면 다음으로 “실제 통화 처리 로직(턴 생성/이벤트 로그 흐름 강화)”를 진행하겠습니다.
