# 답변
로컬스토리지에 auth 키가 없다면 현재 세션이 쿠키 기반입니다. 이 경우 브라우저 콘솔에서 토큰을 읽을 수 없기 때문에 /api/simulate-call이 Authorization 헤더 없이도 동작하도록 서버를 수정해야 합니다.

진행 방법
1) 제가 /api/simulate-call 및 내부 API가 쿠키 세션을 읽도록 수정합니다.
2) 이후 브라우저에서 아래처럼 바로 호출 가능합니다.

```ts
await fetch("/api/simulate-call", {
  method: "POST",
  body: JSON.stringify({ escalated: true })
});
```
