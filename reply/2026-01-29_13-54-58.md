# 답변
해당 위치는 ADMIN 모드에서만 렌더링되는 “Policy JSON” 패널 내부의 두 번째 블록이며, 규칙/템플릿/툴 정책 영역이 포함된 컨테이너입니다. 구현 방식은 다음과 같습니다.

- 조건부 렌더링: `isAdminUser && kbType === "admin"`일 때만 표시됩니다.
- 레이아웃: `div`에 `grid`/`gap-*` 클래스를 적용해 블록 단위로 쌓이는 구조입니다.
- 본문 컨테이너: `rounded-xl border bg-slate-50 p-3`로 카드형 영역을 만들고, 내부에 `select multiple`를 배치했습니다.
- select: `multiple` + `value`를 모든 항목 ID로 고정하여 “항상 전체 선택” 상태를 유지합니다.
- 비편집 처리: `pointer-events-none` + `select-none`로 사용자 입력만 차단합니다(시각적으로는 select 박스로 유지).
- 데이터 소스: `rulePresets`/`templatePresets`/`toolPolicyPresets` 배열이 곧 옵션 렌더링과 JSON 생성의 기준이 됩니다.

해당 부분 코드는 `src/app/app/kb/new/page.tsx`의 ADMIN Policy JSON 섹션에서 확인할 수 있습니다.
