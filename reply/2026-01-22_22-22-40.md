아래 안내는 **mejai.help** 기준입니다. `mejai.hlep`는 오타로 보입니다. 먼저 도메인 이름을 **mejai.help**로 확정해 주세요.

현재 Porkbun 기록:
- ALIAS `@` → `pixie.porkbun.com`
- CNAME `*` → `pixie.porkbun.com`

Railway 연결용으로 이렇게 바꾸면 됩니다 (초등학생 버전):

1) Porkbun 로그인 → 도메인 `mejai.help` → **Manage DNS Records**로 들어가요.
2) **Current Records**에서 아래 2개를 **삭제**해요.
   - ALIAS `@` → `pixie.porkbun.com`
   - CNAME `*` → `pixie.porkbun.com`
3) **새 레코드 추가** 버튼을 눌러서 아래처럼 입력해요.
   - Type: **ALIAS**
   - Host: **(비워둠)**  ← 비워두면 `@`(루트 도메인)예요
   - Answer/Value: **e62658wb.up.railway.app**
   - TTL: **600** (그대로 두면 됩니다)
4) 하나 더 추가해요.
   - Type: **CNAME**
   - Host: **www**
   - Answer/Value: **e62658wb.up.railway.app**
   - TTL: **600**
5) 저장하고 **최대 72시간** 기다려요. 보통은 5~30분 안에 됩니다.

잘 되었는지 확인:
- 브라우저에서 `https://mejai.help` 와 `https://www.mejai.help` 접속이 되면 성공입니다.
- Railway 대시보드의 Custom Domain 상태가 “Verified”로 바뀌면 성공입니다.

만약 Porkbun 화면에 ALIAS가 없고 CNAME만 보이면:
- 알려주세요. 그 경우엔 `@`에 CNAME을 못 넣기 때문에 다른 방법(리다이렉트 등)을 안내해드릴게요.
