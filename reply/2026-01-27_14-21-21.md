# 답변
Deployments 탭에 버튼이 없는 이유는 **이 Cron 서비스에 소스(Repo/Image)가 연결되지 않았기 때문**입니다. 배포를 만들려면 먼저 Source를 연결해야 합니다.

방법 A (권장, 가장 간단): Connect Image로 Node 이미지를 연결
1) `cafe24-refresh-cron` 서비스 → **Settings** 탭
2) **Source** 섹션에서 **Connect Image** 클릭
3) Image에 `node:22-alpine` 입력 → **Connect**
4) 연결 후 **Deployments** 탭으로 이동 → **Create Deployment** 버튼이 생김
5) **Create Deployment** 클릭
6) 성공 후 **Cron Runs** 탭에서 Run now 활성화 확인

방법 B (GitHub Repo 연결)
1) `cafe24-refresh-cron` 서비스 → **Settings** 탭
2) **Source** 섹션에서 **Connect Repo** 클릭
3) `5ys-5y5/mejai` 선택 → **Connect**
4) 연결 후 **Deployments** 탭에서 **Create Deployment** 클릭
5) 성공 후 **Cron Runs** 탭에서 Run now 활성화 확인

중요:
- Cron 서비스에는 **Start command**만 있으면 되므로, 소스는 Image든 Repo든 상관없습니다.
- 이미 설정한 Start command (`node -e ...`)는 그대로 유지하면 됩니다.
