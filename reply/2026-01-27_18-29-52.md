# 질문
"shop_no 목록이 갱신되지 않는 경우, 현재 access_token이 해당 mall_id에 승인된 토큰이 아니라서 그렇습니다. mall_id를 바꾸면 해당 mall_id로 OAuth 승인된 access_token이 필요합니다." 상황을 이해하였습니다.

그렇다면 mall_id를 변경하면 새롭게 해당 mall_id에 대해 https://mejai.help/api/cafe24/refresh를 실행하여 필요한 값을 임시로 얻어올 수 있도록 해주세요.

값을 얻어오고 있을 때는 shop_no 항목과 mall_id 항목에 로딩중이라는 표시를 해야 합니다.

또한 포커스 아웃만으로 탐색을 시작하면 불필요한 요청이 계속 발생할 수 있음으로 mall_id 인풋 안에 돋보기 모양 아이콘을 동그라미 안에 추가하여 버튼을 만들고 
인풋 하단에 탐색 버튼을 클릭하면 shop_no에 해당하는 도메인을 찾을 수 있다는 알림을 추가해야 합니다.

# 답변
요청대로 mall_id 변경 시 자동 refresh를 실행하고, 로딩 상태 표시 및 검색 버튼(돋보기)을 추가하겠습니다. 자동 요청 대신 버튼 클릭으로 탐색이 시작되도록 변경합니다.
