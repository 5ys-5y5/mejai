# 답변
요청사항 반영했습니다.

변경 내용
- `/app/settings` 루트 컨테이너 하단 패딩을 100px로 변경
- scope 멀티셀렉트는 X+N 형식으로 표시
- 입력/셀렉트가 레이블 밖으로 튀어나오지 않도록 `min-w-0` 적용
- 외부 링크 아이콘은 ExternalLink로 유지

하이라이트 관련
- 브라우저 보안상 외부 도메인에 자동 CSS 주입은 불가능합니다. 아이콘 클릭 시 새 탭 오픈 + 클립보드에 하이라이트 스크립트 복사까지는 지원합니다. (콘솔 붙여넣기 필요)

질문 답변
- 네, `client_id/client_secret`은 앱(상점) 단위로 발급되는 값이며, `access_token/refresh_token`은 사용자(상점 관리자) OAuth 동의 결과로 사용자/상점별로 달라집니다.
