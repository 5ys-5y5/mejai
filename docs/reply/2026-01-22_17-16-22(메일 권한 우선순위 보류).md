# 질문
"DNS 레코드 등록"은 porkbun 서비스로 설정하고자 합니다.
이 부분을 좀 더 자세히 알려주세요

# 답변
아래는 Porkbun에서 DNS 레코드를 등록하는 상세 절차입니다. 정확한 값은 사용 중인 메일 발송 서비스(SendGrid/SES 등)에서 제공하는 SPF/DKIM/DMARC 값을 그대로 입력해야 합니다.

1) Porkbun 접속
- Porkbun 로그인 → Dashboard → 도메인 목록에서 `mejai.help` 선택
- 상단의 “DNS” 또는 “Edit DNS”로 이동

2) SPF 레코드 추가 (TXT)
- Record Type: TXT
- Host/Name: @ (또는 빈값)
- Value: 메일 서비스가 제공한 SPF 문자열
  - 예: v=spf1 include:sendgrid.net ~all
- TTL: 기본값(예: 600)

3) DKIM 레코드 추가 (TXT 또는 CNAME)
- 메일 서비스가 제공한 DKIM 레코드 개수만큼 추가
- Record Type: TXT 또는 CNAME (서비스 안내에 따름)
- Host/Name: 예) s1._domainkey 또는 s2._domainkey
- Value: 서비스가 제공한 DKIM 값

4) DMARC 레코드 추가 (TXT)
- Record Type: TXT
- Host/Name: _dmarc
- Value 예시:
  - v=DMARC1; p=none; rua=mailto:dmarc@mejai.help; ruf=mailto:dmarc@mejai.help; fo=1
- 처음에는 p=none으로 시작 후, 발송 안정화 시 quarantine/reject로 강화

5) 저장 후 전파 확인
- Porkbun에 저장 후 10~60분 내 반영
- 메일 서비스의 “Verify” 버튼으로 검증

필요한 정보
- 어떤 SMTP/메일 발송 서비스를 사용하시나요? (SendGrid, AWS SES, Postmark 등)
→ 해당 서비스별로 정확한 SPF/DKIM 값을 안내해 드릴 수 있습니다.
